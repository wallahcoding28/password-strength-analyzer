# 🔐 Password Strength Analyzer

A modern password strength analyzer that combines **Shannon entropy** with **real-world pattern detection** to evaluate password security.

---

## 🚀 Features

* 🔢 Shannon Entropy Calculation (H = L × log₂(R))
* 🧠 Effective Entropy (pattern-adjusted)
* ⚡ Crack Time Estimation (300B guesses/sec GPU model)
* 🔍 Pattern Detection:

  * Dictionary words
  * Keyboard patterns
  * Sequential patterns
  * Repeated characters
  * Substitution attacks
* 📊 Dual Analysis:

  * Brute-force estimate
  * Pattern-aware estimate
* 💡 Security Insight Engine
* 🎨 Modern UI with real-time feedback

---

## 🧠 Key Concept

> High entropy does NOT always mean strong security if predictable patterns exist.
